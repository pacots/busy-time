This folder contains 10 valid 150-job CSV test cases for preemptive busy-time scheduling.
All CSV files use columns: Job, Release, Deadline, processingTime.
Each case has 150 jobs, unique job IDs, unique (Release, Deadline) pairs, processingTime between 20 and 100, valid windows, and max deadline <= 2000.
validation_summary.csv includes quick metrics using capacity g = 2.
